# 🚑 AI Emergency Health Triage System

An intelligent healthcare assistant that analyzes user symptoms and provides real-time risk assessment (LOW, MEDIUM, HIGH), first-aid guidance, and emergency recommendations.

## Features

* 🧠 AI-based symptom analysis
* 🚦 Risk level detection (Low / Medium / High)
* 🩹 First-aid suggestions
* 📞 Emergency response (Call 108)
* 🏥 Doctor and hospital suggestions
* 🎤 Voice input & output
* 🌐 Multi-language support
* ☁️ Firebase chat history storage

##  Tech Stack

* Flask (Backend)
* OpenRouter AI (LLM)
* Firebase Firestore
* HTML, CSS, JavaScript
